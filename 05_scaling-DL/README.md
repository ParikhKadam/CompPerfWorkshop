# Scaling Deep Learning Applications

Author: Sam Foreman ([foremans@anl.gov](mailto:foremans@anl.gov))

We provide examples for:

   2. `Horovod`[`Horovod`](./horovod/torch/README.md)
2. `TensorFlow >= 2`
   1. [`Horovod`](./horovod/tensorflow/README.md)
